Используемые цвета:

Темно-серый (navbar): #262626
Серый (фон шапки): #2a2a2a
Светло-серый (текст): #616161
Красный: #ff6760
Белый (текст): d9d9d9, также используется чисто-белый: #fff
Белый (фон): f8f8f8

Задний фон иконок: 
Оранжевый - #f3b25b
Синий - #5883c0
Циановый - #63bee0
Фуксия - #e9a0c4
Алый - #d35051


Иконки.
Для подключения прописать 
<script src="https://kit.fontawesome.com/64047efea4.js" crossorigin="anonymous"></script>
в head. 

Некоторые иконки помещены в папку icon, раздела image, но многие используются благодаря Font Awesome.

GPS - <i class="fas fa-map-marker-alt"></i>
Телефон - <i class="fas fa-phone"></i>

RSS (значок wi-fi) - <i class="fas fa-rss"></i>
Facebook (буква "f") - <i class="fab fa-facebook-f"></i>
Twitter (буква "t") - <i class="fab fa-twitter"></i> (вместо буквы будет птичка)
Ball (не знаю названия) - <i class="far fa-futbol"></i>
Pinterest (буква "P") - <i class="fab fa-pinterest-p"></i>

Шрифты.
Для подключения шрифтов пропишите в блок head:
<link href="https://fonts.googleapis.com/css2?family=Kaushan+Script&family=Lato:ital,wght@0,300;0,400;0,900;1,300&display=swap" rel="stylesheet">

В CSS:
font-family: 'Kaushan Script', cursive;
либо,
font-family: 'Lato', sans-serif;
в зависимости от того, какой шрифт нужен.
Первый шрифт используется только для логотипа в шапке. В остальном используется второй шрифт.

Жирность текста регулируйте изменением свойства font-weight
300 - для Light-жирности,
400 - для Regular-жирности, 
900 - для Black-жирности.

Курсивное начертание можно сделать свойством font-style: italic;
